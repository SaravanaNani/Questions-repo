### 4️⃣ Monitoring Tools – Prometheus & Grafana

“Prometheus is used for collecting metrics from different systems using exporters like node exporter, kube-state-metrics, and cAdvisor for container metrics. It scrapes metrics at regular intervals and stores them in a time-series database. Grafana is used to visualize these metrics through dashboards. This helps monitor cluster health, resource usage, and application performance.”
